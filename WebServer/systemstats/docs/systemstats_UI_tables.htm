Tabellförklaring

Startad = Datum då Windows startades.
= Tid då Windows startades.
Inloggad = Datum då senaste inloggning gjordes.
= Tid då senaste inloggning gjordes.
Användare = Visningsnamnet för användaren som senast loggade in på datorn.
AD-konto = Användarens inloggningskonto i domänen.
= Användarens telefonnummer (extra modul). Innehåller även signatur för vilken tekniker som markerat en dator som "i lager".
Datornamn = Datorns hostname.
Modell = Modellnamn på datorn.
Serienummer = Datorns serienummer.
OS = Datorns operativsystem.
MB kvar C: = Ledigt utrymme på C-disken i Megabyte.
Installerad = Datum då operativsystemet installerades.
Office = Version av Microsoft Office som är datorn kör.
= Datorns totala mängd arbetsminne (RAM).
CPU = Datorns processor. Visar modell och klockfrekvens.
= Processorns totala antal kärnor.
= Antal logiska processorer (trådar).
Kommentar = Fritextkommentar om datorn.
